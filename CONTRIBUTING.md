# Contributing to OneAI 🔥

Makasih udah tertarik bantu OneAI! Keluarga Termux harus saling bantu.

## Cara Kontribusi

**1. Lapor Bug / Request Fitur**
- Buka Issues: https://github.com/unlimited48/OneAI/issues
- Cek dulu udah ada yang laporin belum
- Pake judul yang jelas: "Bug: Error pas install di Android 11 Samsung A02"
- Kasih detail: versi Termux, Python, log error

**2. Pull Request**
1. Fork repo ini
2. Bikin branch baru: git checkout -b fitur-model-baru
3. Commit rapi: git commit -m "Tambah support model Claude"
4. Push: git push origin fitur-model-baru
5. Buka PR ke branch main OneAI

**3. Aturan Kode**
- Pake Python 3.10+
- Kasih comment kalo logikanya ribet
- Test dulu di Termux sebelum PR
- Jangan masukin API key/token ke kode

**4. Yang Bisa Dibantuin**
Cek label good first issue di Issues. Contoh:
- Tambah model AI baru dari OpenRouter
- Benerin typo di README
- Optimasi speed OneAI
- Bikin dokumentasi Bahasa Indonesia

**5. Gaya Komunikasi**
Santai aja, tapi tetap respek. Kita keluarga di sini.

Udah gitu aja. Nanya-nanya boleh DM atau buka Issue.
